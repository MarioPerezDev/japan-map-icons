How to use →

  • Each tile is a drop zone. Drop a PNG on it (or click to pick a file). The drawing's opaque bounding box is auto-detected and centered.
  • Hover a populated tile for Download PNG, Download SVG, and Replace. PNGs export at 512×512.
  • Use the With circle / Icon only toggle in the toolbar to choose whether downloads include the colored container or just the line drawing on transparent background.
  • Drawings live in this browser session — your PNGs aren't uploaded anywhere. Use Download all to grab the whole set as a single zip.
  • Per the visual identity guidelines: dark line on transparent background. The category color comes from the container fill — drawings are placed as-is, no recoloring.
